Game Idea


Title

The title for this web based game is "Forever Falling", we feel this title is justified from the goal of the game.

Genre

The genre of this game is classed as "Indie Adventure".
Indie: Is short for independent developer, this is developers that make games without backing of any third-party publisher.
Adventure: Is a genre

Concept

The object of the game is to fall forever without hitting any obstacles
The longer you fall your speed will increase and you will gain points.
There is some power-ups features that can assist your travels along the way.

Game Overview


Core Objective

The goal is to fall as long as possible, tallying up points and avoid hitting any object that will end their fall and the current game.
Players are encouraged to use the abilities within the scene to aid their fall. However, not all abilities will be beneficial and some thought process must come in when the occasion arises.

Expected Experience

Players are expected to have fun and feel a sense of laughter from the game.
Players are also to feel a little adrenaline rush from the speed at which the game can be played at.

Gameplay Theme

The games base theme is set in the sky with clouds and floating objects that the player must avoid.

Game Structure



Features Implemented

In Forever Falling, there is many features that we call in-game abilities to aid your fall.
These abilities can randomly appear on the scene as you fall so there is always a 'new game' experience whenever you play.
Abilities:
Speed Up - The speed up ability will increase the speed in which the player falls at. This can be a good advantage in the early rounds of the game to obtain points faster.
Speed Down - The speed down ability will decrease the speed in which the player falls at. This can be a good advantage in later stages of the game to give the player time to compose their self.
Double Points - The double points ability will double the players points for a limited amount of time. There is no disadvantages to this ability and we encourage the player to collect these.
Spawn More Objects - The spawn more objects ability will add more objects in the game. This is a disadvantage feature to balance the game, and we strongly advise to avoid this ability. Every one of these collected adds one more spawner to the current game. Resulting in one solid object being spawned as you all on top of the base amount.

Level Design


Scene Description

The scene consists of a never ending blue sky, which will have clouds splitting up the blue. These clouds will not be interactable. The scene as you progress down will randomly spawn objects such as barrels and wooden planks that the player will have to avoid in order to progress down the scene and obtain power ups n points. As the player progresses down the scene the ability features will start appear such as; slow down ability, speed up ability, double points ability, spawn more objects ability and bombs.

Intended Gameplay Sequence

As the player falls, he or she will have the avoid any collisions with solid objects. If there is any collisions with a solid object the game will end and the player will no longer be falling.
There will be power ups and power downs availible for the falling player to take on board. However, not all of these items are a good thing, there are some bad eggs within that basket.
The longer the player falls the faster he or she will become and they will gather a better score.

Phaser

We developed this game using Phaser, "a fast, free and fun open source framework for canvas and WebGL powered browser games.

What is Phaser?

Phaser is an HTML5 game framework which aims to help developers make powerful, cross-browser HTML5 games really quickly and, unlike some others, has solely been built to work with the mobile browsers. The only browser requirement is the support of the canvas tag. It also borrows a lot from Flixel.

Behind The Idea

There wasn't a lot of research put into the idea of the game, as the idea was there for a while. However, when obtaining facts for this documentation I put research into similar games and competitors. I used Phasers website, the framework behind this game, to research similar games and see how other people implemented their ideas. There wasnt much I found that related to the idea I had, although there were games with similar controls and features like Starstruck & Invaders.

Inspiration

There were numerous sources of inspirations when thinking about this 2D web game. However, the main source of this expression came from a dark reality, a time were I felt that I was falling from a high state of mind and seemed to last forever. Wishing there were magical abilities to slow everything down, just a little and gain a few seconds. Allowing the return of having control of the path ahead. Games like Portal helped me ground my feelings and turn them into an playable expression that I hope people will enjoy when playing "Forever Falling".
